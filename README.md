# ☕ Starbugs Cucumber Automation

Projeto de automação de testes BDD para o site Starbugs, focado em práticas de QA Moderno.

## 🛠️ Tecnologias
- **Ruby** 3.4
- **Cucumber** (Escrita de cenários BDD)
- **Capybara & Selenium** (Interação com navegador)
- **RSpec** (Asserções e validações)

## 🏗️ Arquitetura
O projeto utiliza o padrão **Page Objects** para separar a lógica de negócio (cenários) da interação com os elementos da tela, garantindo uma manutenção muito mais simples.

## 🚀 Como rodar
1. Instale as dependências: `bundle install`
2. Execute os testes: `bundle exec cucumber`
